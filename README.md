# Registration REST API with Node.js, ExpressJS, Mongoose & jetwebtoken & dotenv
# Features:-
Register a user</br>
Verify user's email address</br>
Send forgot password email</br>
Reset password</br>
Login</br>
Access token</br>
Refresh tokens</br>
